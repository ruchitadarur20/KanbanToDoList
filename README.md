# KanbanToDoList
A web-based Kanban To-Do List for task management with prioritization and animations


Kanban To-Do List
Project Description
The Kanban To-Do List is a dynamic web-based application designed to help users organize and manage tasks effectively using the Kanban methodology. This tool allows users to visually categorize tasks into three stages: To-Do, In Progress, and Done.

With an intuitive interface and responsive design, users can easily add tasks, assign priorities, and move tasks across different stages as they progress. The project also includes a task priority system and supports local storage, ensuring tasks persist even after refreshing the browser.

#Features
Task Management: Add, delete, and manage tasks.
Kanban Board: Visually organize tasks into To-Do, In Progress, and Done columns.
Prioritization: Assign priority levels (High, Medium, Low) to tasks with distinct colors for easy identification.
Local Storage Support: Automatically saves tasks locally, ensuring they remain after a browser refresh.
Interactive Animations: Smooth animations for task transitions and hover effects for enhanced usability.
Responsive Design: Fully functional on desktops and mobile devices.
Date Tracking: Track start and end dates for each task.
Custom Styling: Modern, clean, and vibrant design with customizable color themes.

#Technologies Used
HTML5: Markup language for structuring the application.
CSS3: Styles for layout, colors, animations, and responsiveness.
JavaScript (ES6): Functionality for task manipulation, animations, and local storage integration.

#How to Use
Add Tasks: Use the input fields in the To-Do column to add tasks along with start dates and priorities.
Move Tasks: Move tasks between To-Do, In Progress, and Done columns by clicking the respective action buttons.
Delete Tasks: Remove tasks using the delete button.
Save Progress: All changes are saved automatically using local storage.
